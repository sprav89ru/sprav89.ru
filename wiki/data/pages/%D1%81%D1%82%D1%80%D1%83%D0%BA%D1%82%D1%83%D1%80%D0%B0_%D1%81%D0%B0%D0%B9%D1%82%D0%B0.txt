===== Структура сайта =====
Сайт расположен по адресу http://sprav89.ru/ \\ 
(справочник, 89 - региональный код ЯНАО. Многие, по крайней, мере в г. Новый Уренгой используют эту комбинацию цифр для идентификации) \\ 
Локальная версия располагается в Каталог Apache/htdocs/sprav89.ru\\ 
Для выгрузки / загрузки на Github — в каталоге Apache/htdocs/sprav89.git\\ 
Пока основное отличие между ними — последний содержит архив MySQL (уменьшенный, предназначенный только для отладки) и файл login.php для открытого доступа.\\ 

Основное содержимое сайта: \\ 
  * левый сайдбар (sidebar1.html); \\ 
  * основной контент (загружается в зависимости от параметра $_GET, по умолчанию (первоначальная загрузка) — main.html); \\ 
  * правый сайдбар (sidebar2.html). \\ 

Весь алгоритм загрузки описан в файле [[файл_index.html|index.html]]. \\ 
Если файл содержит в основном разметку, он имеет расширение html, если код — php или js. \\ 
Файл, если не указано обратного, находится в корневом каталоге sprav89.ru. \\
